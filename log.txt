Error ini terjadi karena ada konflik kompatibilitas antara PyTorch 2.4.1+cpu dengan torch-directml. Mari kita perbaiki dengan beberapa solusi:
🔧 Solusi 1: Clean Install torch-directml (Recommended)
bash# Uninstall PyTorch yang ada
pip uninstall torch torchvision torchaudio

# Install torch-directml (ini akan install PyTorch yang kompatibel)
pip install torch-directml

# Verify installation
python -c "import torch_directml; print('DirectML installed successfully')"
🔧 Solusi 2: Downgrade PyTorch
Jika solusi 1 tidak berhasil, coba downgrade PyTorch:
bash# Uninstall current PyTorch
pip uninstall torch torchvision torchaudio torch-directml

# Install specific compatible versions
pip install torch==2.1.0 torchvision==0.16.0 torchaudio==2.1.0 --index-url https://download.pytorch.org/whl/cpu
pip install torch-directml
🔧 Solusi 3: Buat Environment Baru
Cara paling aman adalah membuat environment baru:

conda create -n vit-amd python=3.9
conda activate vit-amd

pip install torch-directml
pip install transformers datasets accelerate
pip install scikit-learn matplotlib seaborn pillow numpy pandas

Batch size akan otomatis dikurangi (4 vs 10)
Tetap bisa training ViT dengan baik
